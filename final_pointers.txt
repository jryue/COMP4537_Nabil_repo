Starting the mongoose database
1. click Windows -> type "ubuntu" -> click the program
2. opens a terminal -> type "sudo service mongodb start"
3. prompts for user password -> pw: Jr25360912
4. type "mongo"
5. "use test"
6. done.

2. or, create one -> 27017 is the default
